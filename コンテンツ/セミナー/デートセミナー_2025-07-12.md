## 2. ゴール設定・逆算思考

**距離ゲージ**という概念を持つ（相手との距離感の数値化）。

```
0 ----30---- 50---- 80---- 100
     ↑       ↑      ↑
  デートOK  2回目  GET可能
```

- 電話：30点
- 初回デート：50点
- 2回目デート：80点
- ※減点・加点により変動

**重要ポイント**
- 距離が縮まっていないのにアプローチするからミスる
- 距離が縮まっているのにアプローチしないから愛想尽かされる
- マッチしてる時点である程度の脈はある → どんどん攻めよう

**ゴール設定**
- 電話 → デートに行くため
- 初回デート → 2回目に行くため
- 2回目デート → GETするため

## 3. 店選び・動線

**初回デート設計**
- 目的：次回デートに誘う
- 昼カフェ（60〜90分）
- 解散予定をあらかじめ決めておく
- 「デートの内容より、あなたと話すこと」を目的とする

**NG店舗**
- 映画館・動物園・ディズニー（それ自体が目的になる）→ 付き合ってからでよい

**お店選びチェック**
- 雰囲気が良い（高級すぎ・チェーン NG）
- 横並び or L字席
- 隣の席と距離がある
- 店内の音量が適切
- ホームグラウンド（勝手がわかる場所）
- 立地が良い
- 予約必須・待ちが発生しない

**お店を見つける3ステップ**
1. 「地域＋目的」でInstagram検索（◯選系が便利）
2. Googleマップにピンを立てる
3. 実際に現地に下見に行く

**2軒目**は事前に近隣スポットを押さえておく。

## 5. デート立ち回り

### 1stコンタクト
> 合流して10秒以内にオスとしてありかなしか判断される

電話で合流→目を見る→笑顔→褒め→自信・余裕

### 主導権を握る方法
道路で先を歩く、テンション、メニューを見せる、リアクション、QRコード、声・表情・ノンバーバル、おしぼりを渡す、話題展開、店員さんを呼ぶ、質問・深掘り、サラダの取り分け、切り上げ

### エスコート（レディーファースト）
| 場面 | 行動 |
|-----|------|
| 歩行時 | 車道側を歩く、速度を合わせる |
| エスカレーター | 男が下 |
| エレベーター | 女性が先 |
| 席 | 椅子を引く、メニューは女性向き |
| 注文 | 男が決める、店員は男が呼ぶ、おしぼりは先に女性へ |
| 会計 | トイレの間に済ませる |
| 解散 | 改札まで女性を送る |

### 感情を動かす
嬉しい・楽しい・悲しい・辛い・苦しい・うざい・キモい  
→ 共有する・共感する・妄想させる・思い出させる  
→ **ドキドキと安心感の緩急（ギャップ）**を作る

### 特別感を出す
- 「ただ友達みたいな会話」ではNG
- 相手に興味を持ち、相手を理解する（自分→相手にベクトルを向ける）
- 「君が好き」ではなく「こんな人が好き」で該当ゾーンを伝える

## 6. 会話

**距離ゲージを縮める4つのフェーズ**
1. アイスブレイク
2. 価値証明
3. ヒアリング
4. アプローチ

会話の順番：仕事の話 → 趣味の話 → 過去の話 → 恋愛の話

## 7. 会計と解散

- 必ず次回のデートに誘うこと
- 満足させずに余韻を残す
- スマートに「またね」
- **原則奢る**（カフェで2〜3,000円）
- いいなと思わなかった子は全部奢らず多めに出すのもOK

## 9. 2回目デートまで（熱量を落とさない）

LINE4パターン
1. 今日あったことを共有する（食べたもの・行った場所・見た動画など）

3. 相手のおすすめの感想を伝える（映画・カフェ・グルメなど）
4. 適当な雑談（知性・ユーモア・ノリで価値証明）

## 10. 2回目デート

**2回目デート設計**
- 目的：告白 or お持ち帰り
- 夜居酒屋 → 個室・バー・シーシャ等（90〜120分）
- ゴールを決めておく

**心の距離を縮める**
未来トーク、究極理解、価値証明、好意の匂わせ、安心感と特別感、キレイごとトーク

**身体の距離を縮める（りょうメソッド）**
一か八かではなく確認作業＝スタンプラリー  
グラスを近づける → 身体を近づける → 足をくっつける → 手を繋ぐ → 肩を寄せる → 目を見つめる → キスをする（スモールステップで）

## 11. GETしたあと

**非日常・非効率なことを楽しもう**
- 非日常：国内・海外旅行、予約の取れないレストラン、ディズニー・USJ、花火大会、クリスマスマーケット
