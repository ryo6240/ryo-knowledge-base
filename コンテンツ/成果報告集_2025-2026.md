## 成功パターンのまとめ

### 共通点

| 項目 | 傾向 |
|------|------|
| 使用アプリ | with が最多、次いでPairs。東カレ・Omiai も |
| 告白タイミング | 2回目デートが最多。1回目や3回目も |
| 告白場所 | シーシャ個室・公園・夜景スポット・店の外が定番 |
| 電話回数 | 0〜6回（2回前後が多い） |
| サロン入会〜彼女 | 1〜2ヶ月が多い。中には1日〜1週間も |
| 人生初彼女の報告 | 非常に多い（20〜30代での初彼女も珍しくない） |

### 成功した告白フレーズ例

- 「○○ちゃん、これからも一緒にいたい。俺と付き合おう」
- 「価値観めっちゃ合うし、お互いのことをもっと知るために付き合お！」
- 「一緒にいて楽しいし、もっと思い出作りたいから、付き合おうか？」
- 「てかもう好きだわ！俺の彼女になってよ」
- 「俺と付き合おう」（シンプルに）
- 「付き合ったらこういうことしてみたい」→未来トークから自然に

## 成功に共通するサロンの活用法

1. **デート報告・成果報告を読んでモチベ維持**  
2. **りょうメソッドの実践**（グラスを近づける・肩タッチ・足を揃える）  
3. **オフ会参加で実践経験・仲間からの刺激**  
4. **写真撮影（けいとさん）でプロフィール強化**  
5. **個別チャンネルでりょうさん・えいすけさんへの相談**  
6. **電話音声共有チャンネルの活用**（何度も聴いてメモ取り）

## withの足跡営業に関する注意点（失敗談より）

- スマホはかならず**アプリ版でログイン**すること（ウェブ版ではスワイプ連打ができない）  
- PC（ふむふむくん）＋スマホアプリの2デバイス体制が正しいやり方  
- 登録後24時間以内の足跡に無料いいねが送れる仕組みを活かす
