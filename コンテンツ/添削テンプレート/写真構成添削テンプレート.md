# 写真構成 添削テンプレート

> 用途：マッチングアプリ写真構成フィードバックの下書き生成用テンプレート  
> 参照基準：`/エージェント/カスタマーサポート/添削基準/写真構成添削基準.md`

---

## 使い方

1. サロン生から現在の登録写真（全枚数）を受け取る
2. 添削基準に照らして各枚数を確認
3. 以下テンプレートで下書きを生成し、りょうさんへ提出

---

## 添削フィードバック（下書き）

```
【写真構成 添削フィードバック】

■ アプリ
{with / Pairs / 東カレデート / タップル / その他}

■ 全体の印象
{全体的な評価を1〜2文で。清潔感・信頼感・余裕感が伝わるかを軸に}

■ 1枚目（メイン写真）
良い点：{具体的に}
改善点：{具体的に}

■ 2枚目以降
{各枚数ごとに「内容・良い点・改善点」を簡潔に}
2枚目：
3枚目：
4枚目：
5枚目以降：

■ 写真構成の改善提案
現状：{現在の構成}
推奨：
  1枚目：顔がはっきり見えるメイン写真
  2枚目：全身または上半身・ファッションが伝わる写真
  3枚目：趣味・生活感が伝わる写真（旅行・カフェ・スポーツ等）
  4枚目：別角度・別表情
  5枚目以降：アクティブさ・多面性が伝わる写真

■ 差し替え推奨
{差し替えたほうがいい写真番号と理由}

■ 撮影で追加したい写真
{今後の撮影で追加すべきシーン・構図}

■ 総合判定
{このまま運用OK / 一部入れ替え推奨 / 全体的に撮り直し推奨}
```

---

## チェックリスト（添削前に確認）

### 1枚目（必須項目）
- [ ] 顔がはっきり見えるか
- [ ] 自然な笑顔 or 余裕感のある真顔か
- [ ] 背景がシンプル・おしゃれか（生活感・雑然とした背景でないか）
- [ ] 明るく・自然光で撮れているか
- [ ] 服装に清潔感があるか

### 全体構成
- [ ] 複数枚で異なる表情・シーンが見えるか
- [ ] 全身・上半身が伝わる写真があるか
- [ ] 趣味・アクティブさが伝わる写真があるか
- [ ] サングラス・逆光・ピンボケの写真がないか
- [ ] 他人が写り込んでいる写真がないか

---

## アプリ別 重視ポイント早見表

| アプリ | 1枚目 | 全体の雰囲気 |
|--------|-------|------------|
| with | 笑顔・親しみやすさ | 明るく清潔感。絵文字的要素OK |
| Pairs | 誠実さ・自然な笑顔 | 落ち着いた雰囲気・真剣さ |
| 東カレ | 大人っぽさ・センス | 洗練された場所・服装。絵文字的要素NG |
| タップル | 元気・若々しさ | アクティブなシーンも有効 |
