# 自己紹介文 添削テンプレート

> 用途：添削フィードバックの下書き生成用テンプレート  
> 参照基準：`/エージェント/カスタマーサポート/添削基準/自己紹介文添削基準.md`

---

## 使い方

1. サロン生から自己紹介文テキスト＋アプリ名を受け取る
2. 添削基準に照らして確認し、以下テンプレートに当てはめる
3. りょうさんへ下書きとして提出（Discordへの送信はりょうさん確認後）

---

## 添削フィードバック（下書き）

```
【自己紹介文 添削フィードバック】

■ アプリ
{with / Pairs / 東カレデート / タップル / その他}

■ 全体の印象
{全体的な評価を1〜2文で。ポジティブな点から始める}

■ 修正が必要な箇所

① {該当箇所の引用}
→ 修正案：{修正後のテキスト}
理由：{なぜ修正するか}

② {該当箇所の引用}
→ 修正案：{修正後のテキスト}
理由：{なぜ修正するか}

（修正箇所なければ「修正不要」と記載）

■ 良かった点
- {そのまま使えるポイント1}
- {そのまま使えるポイント2}

■ 添削後の全文
---
{添削後の自己紹介文全文をここに記載}
---

■ 総合判定
{OK / 一部修正 / 大幅修正推奨}
```

---

## チェックリスト（添削前に確認）

- [ ] 冒頭：アプリを始めた理由になっているか（「初めまして〜」「足跡〜」で始まっていないか）
- [ ] 冒頭：「気になった方だけにいいねしています」などの選り好み表現がないか
- [ ] 構成：「仕事→性格→趣味→恋愛観」の順になっているか
- [ ] 仕事：職種＋きっかけ＋向上心まで書かれているか
- [ ] 趣味：羅列でなく各趣味に1文のエピソードがあるか
- [ ] 趣味：共通の話題になりやすいものが先頭にあるか
- [ ] 末文：電話誘導フレーズが入っているか
- [ ] 末文：「足跡は追いきれてません」「まずはメッセージから〜」等の受け身表現がないか
- [ ] 末文：「ここまで読んでいただきありがとうございます」を削除したか
- [ ] 締め文：「少しでもいいなと思ったら / 気軽にいいねお願いします👍🏻」になっているか
- [ ] アプリ：東カレの場合は絵文字NGになっているか

## よく使う定型フレーズ集

### 冒頭（2パターン）
```
【パターンA】
友人がこのアプリで素敵な方を見つけていて、羨ましいなと思い始めました！
```
```
【パターンB】
仕事や趣味、友達との時間も大切にしつつ、支え合い・楽しんでいける方と巡り合えればと思い登録しました！
```

### 末文（3パターン）
```
【標準】
メッセージだと分からないことも多いので、電話をしてからお会いしたいです。
少しでもいいなと思ったら / 気軽にいいねお願いします👍🏻
```
```
【強め】
メッセージだと正直何も分からないので、早めに電話をしてからお会いしたいです📞
少しでもいいなと思ったら / 気軽にいいねお願いします👍🏻
```
```
【リアル誘導】
気が合えばまずはカフェでお茶しましょう。
少しでもいいなと思ったら / 気軽にいいねお願いします👍🏻
```
