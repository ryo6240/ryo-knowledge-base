# 服装 添削テンプレート

> 用途：服装フィードバックの下書き生成用テンプレート  
> 参照基準：`/エージェント/カスタマーサポート/添削基準/服装添削基準.md`

---

## 使い方

1. サロン生から全身着用写真（複数枚）を受け取る
2. 添削基準（体型補正4ポイント・チェック基準）に照らして確認
3. 以下テンプレートで下書きを生成し、りょうさんへ提出

---

## 添削フィードバック（下書き）

```
【服装 添削フィードバック】

■ 用途
{マッチングアプリ写真用 / デート当日用}

■ 全体の印象
{全体的な評価を1〜2文で}

■ 体型補正チェック
- 肩幅：{大きく見えているか / 要改善}
- 腰位置：{高く見えているか（タックイン等）/ 要改善}
- パンツ×靴の色：{合っているか / 要改善}
- アウター丈：{短め（腰丈〜ヒップ上）か / 要改善}

■ 候補の中からおすすめ2パターン
パターンA：{写真番号または説明}
パターンB：{写真番号または説明}

■ 改善提案
- {具体的な改善点1（アイテム名を明記）}
- {具体的な改善点2}

■ アクセサリー確認
「腕時計・ブレスレット・ネックレスはお持ちですか？お持ちでしたらお送りください。」

■ 総合判定
{このまま撮影OK / 一部修正して撮影 / 買い足し・コーデ同行推奨}
```

---

## チェックリスト（添削前に確認）

- [ ] 体型補正①：肩幅が大きく見えるシルエットか
- [ ] 体型補正②：腰の位置が高く見えるか（タックイン・ハイウエスト）
- [ ] 体型補正③：パンツと靴の色が合っているか
- [ ] 体型補正④：アウターが短め丈（腰丈〜ヒップ上）か
- [ ] 清潔感：シワ・毛玉・汚れがないか
- [ ] サイズ感：ダボダボ・パツパツになっていないか
- [ ] 色：ベーシックカラー中心で2〜3色以内か
- [ ] 素材：テカテカポリエステル・ヨレたニットを使っていないか
- [ ] 季節感：季節に合った素材・アイテムか

---

## コーデ方向性の設定（最初に決める）

| 方向性 | 特徴 | 向いている人 |
|-------|------|------------|
| カジュアル系 | 素材感・遊びのある服 | 顔立ちが柔らかい・若く見える |
| キレイめ系 | テーラード・スラックス中心 | 大人っぽい・落ち着いた雰囲気 |
| 韓国系 | トレンド感・オーバーシルエット | スタイリッシュ・細身 |

---

## 撮影当日の持参指示（確定後に送る）

```
撮影当日は以下をお持ちください：

・コーデAセット一式（上下・靴・アクセ）
・コーデBセット一式（上下・靴・アクセ）
・インナー：白T＋黒T両方
・シワの最終チェックをお願いします
```

---

## 手持ちが不足している場合

```
手持ちの服が足りない場合は、一緒に買いに行きましょう！
（ZARA / H&M / ユニクロ / GU）
撮影日と同日にまとめて対応もできます。
```
