# 髪型 添削テンプレート

> 用途：髪型フィードバックの下書き生成用テンプレート  
> 参照基準：`/エージェント/カスタマーサポート/添削基準/髪型添削基準.md`

---

## 使い方

1. サロン生から現在の髪型の写真を受け取る（写真がない場合は促す）
2. 添削基準に照らして確認
3. 以下テンプレートで下書きを生成し、りょうさんへ提出

---

## 添削フィードバック（下書き）

```
【髪型 添削フィードバック】

■ 全体の印象
{全体的な評価を1〜2文で}

■ 良い点
{そのままキープしてほしい点}

■ 改善提案
- {具体的な改善点1}
- {具体的な改善点2}

■ 目指すスタイルの方向性
{センターパート / アップバング / ツーブロック系 / その他}

■ 美容室への指示例
「{美容師に伝える具体的な言葉}」

■ ヘアセット手順
1. ドライヤー（最重要）
2. アイロン（必要な場合）
3. ワックス
4. スプレー

■ 眉毛チェック
{整っているか / 眉毛サロン推奨}

■ 参考動画
{該当する場合は以下から選んで添付}
- アップバング：https://youtu.be/Cuev5buso3U
- ドライヤー：https://youtu.be/cSaC-oG7YTM
- アイロン・ワックス・スプレー：https://youtu.be/bW9S7c7uj0s

■ 総合判定
{OK / 一部修正 / カット推奨}
```

---

## チェックリスト（添削前に確認）

- [ ] 清潔感：ボサボサ・油っぽさ・寝ぐせがないか
- [ ] スタイリング：ノーセットになっていないか（ワックスのみも要注意）
- [ ] カラー：自然な黒〜ブラウン系か（金髪すぎ・不自然なカラーでないか）
- [ ] 長さ：伸ばしっぱなし・手入れ不足でないか
- [ ] 眉毛：髪型と一緒に整っているか
- [ ] 撮影との整合性：写真とリアルで同じ髪型・セットができるか

---

## 推奨スタイル早見表

| スタイル | 特徴 | 向いている人 |
|---------|------|------------|
| センターパート | 清潔感・トレンド感 | 幅広く対応可 |
| アップバング | 爽やか・清潔感・額を見せる | 練習が必要。早めに着手 |
| ツーブロック系 | サイドすっきり・全体が軽く見える | 幅広く対応可 |

## 撮影前の準備指示

```
撮影前に以下をお願いします：

・撮影2〜3日前：美容室でカット＆眉毛を整える
  （推奨：オーシャントーキョー / リップス）
・撮影当日朝：ヘアセットを再現する
  （ドライヤー→アイロン→ワックス→スプレーの順）
・シワの最終確認も合わせてお願いします
```
