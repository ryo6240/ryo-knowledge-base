# マッチングアプリ 写真構成添削基準

> 用途：カスタマーサポートエージェントが写真構成の下書きフィードバックを生成するための基準  
> 最終更新：2026年4月19日

---

## 写真構成の基本方針

- **清潔感・信頼感・余裕感**の3つが伝わるかどうかが最重要
- アプリごとに求められる雰囲気が異なる（例：東カレ＝大人っぽさ、タップル＝明るさ）
- 写真は1枚目が命。マッチング率を大きく左右する

---

## 写真1枚目（メイン写真）の基準

| 項目 | OK | NG |
|------|----|----|
| 表情 | 自然な笑顔 or 余裕感のある真顔 | 不自然な作り笑い、暗い表情 |
| 背景 | シンプル・おしゃれな場所（カフェ、屋外等） | 生活感のある部屋、雑然とした背景 |
| 顔の見え方 | 顔がしっかり見える | 遠すぎて顔がわからない、サングラス |
| 明るさ | 明るい・自然光 | 暗い・室内光で影がある |
| 服装 | 清潔感のある着こなし | よれた服、カジュアルすぎる |

---

## 写真構成の全体パターン（推奨）

| 枚数 | 内容 |
|------|------|
| 1枚目 | 顔がはっきり見える・清潔感のあるメイン写真 |
| 2枚目 | 全身または上半身。ファッション・スタイルが伝わる写真 |
| 3枚目 | 趣味・生活感が伝わる写真（旅行・カフェ・スポーツ等） |
| 4枚目 | 別角度・別表情。笑顔か余裕感のある表情 |
| 5枚目以降 | アクティブさ・多面性が伝わる写真 |

---

## アプリ別 写真方針

| アプリ | 求められる雰囲気 | 注意点 |
|--------|----------------|--------|
| with | 明るく親しみやすい | 清潔感優先。笑顔が効果的 |
| Pairs | 誠実さ・真剣さ | 落ち着いた雰囲気。自然な笑顔 |
| 東カレデート | 大人っぽさ・センス | 洗練された場所・服装。絵文字的な要素を排除 |
| タップル | 明るさ・若々しさ | 元気・アクティブなシーンも有効 |

---

## 写真ごとのチェックリスト

```
□ 顔がはっきり見えるか
□ 背景が清潔・おしゃれか
□ 服装に清潔感があるか（シワ・よれがないか）
□ 表情が自然か（緊張・引きつりがないか）
□ 全体的に明るく撮れているか
□ 複数枚で別の表情・シーンが見えるか
□ NGアイテムがないか（サングラス、逆光、ピンボケ等）
```

---

## フィードバックテンプレート

```
【写真構成 添削フィードバック（下書き）】

■ 全体の印象
{全体的な評価を1〜2文で}

■ 1枚目について
{良い点・改善点を具体的に}

■ 2枚目以降について
{構成上の改善提案}

■ 改善提案
- {具体的なアクション1}
- {具体的なアクション2}
- {撮り直し推奨の場合はその旨}

■ 総合判定
{OK / 一部修正 / 撮り直し推奨}
```

## りょうコーチの判断傾向（実例より）

- 「写真より可愛くて安心した」という言葉をデートで使わせない → 写真と実物のギャップをなくす
- 季節が変わったタイミング（春・夏）での撮り直しを推奨
- 写真に慣れること自体が重要。「撮り慣れる」ことで清潔感・印象が上がる
- プロフィール写真のコメント（一言）は「話しかけやすい言葉＋絵文字1〜2個」が基本

---

## 要確認・エスカレーションの基準

- 実際の写真データが添付されている場合 → 画像を見て具体的にフィードバック
- 写真の説明のみの場合 → 「実際に写真を送ってもらえますか？」と促す
- 撮り直しが必要な場合 → 「撮影のポイント」を具体的に提示する
