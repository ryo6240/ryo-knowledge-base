# 自己紹介文 添削基準

> 用途：カスタマーサポートエージェントが自己紹介文の添削下書きを生成するための基準  
> 出典：りょうコーチ添削実例（岡さん・けいさん・ひろさん）  
> 最終更新：2026年4月19日

---

## 添削の基本方針

1. **冒頭文でアプリを始めた理由を自然に書く**
2. **受け身・消極的な表現を削除する**
3. **セクション順は「仕事→趣味→性格→恋愛観」が基本**
4. **「足跡追いきれてません」「気軽にいいね」などの受け身表現はNG**
5. **全体的にシンプル・自然体で書く**

---

## セクション別チェック基準

### 冒頭文
| OK | NG |
|----|-----|
| アプリを始めた自然な理由（「友人がここで素敵な出会いをしたので」） | 「気になった方に足跡つけてます」 |
| 前向きな一言（「どうぞよろしくお願いします！」） | いきなり自己PRから始まる |

### 仕事・職業
| OK | NG |
|----|-----|
| 職種＋やりがいを1〜2文 | 「都内で働いています」（曖昧すぎる） |
| 具体的・信頼感のある書き方 | 職場名を細かく書きすぎる |

### 趣味・好きなこと
| OK | NG |
|----|-----|
| 共通の話題になりやすいもの（カフェ・旅行・スポーツ等） | 趣味の羅列だけで感情がない |
| 「なぜ好きか」を1文添える | マニアックすぎる趣味のみ |

### 性格
| OK | NG |
|----|-----|
| 具体的なエピソードや特徴 | 「穏やかです」だけで終わる |
| 第三者から言われることを引用（「〜と言われます」） | 自己申告が強すぎる |

### 好きなタイプ・恋愛観
| OK | NG |
|----|-----|
| 相手が「自分かも」と思えるような描写 | 「笑顔が素敵な人」（抽象的すぎる） |
| お互いを尊重する表現 | 「理想が高すぎる」と思われる表現 |

### 締め文
| OK | NG |
|----|-----|
| 「お気軽にメッセージいただけると嬉しいです」 | 「足跡は追いきれてません」 |
| 軽い一言で前向きに締める | 「いいねください」「マッチしましょう」（積極的すぎる） |

## アプリ別 注意点

| アプリ | 特徴 | 注意点 |
|--------|------|--------|
| with | 比較的カジュアル | 絵文字少量はOK。明るい文体 |
| Pairs | 真剣度が高い | 誠実さ・落ち着いた文体。絵文字控えめ |
| 東カレデート | ハイスペック層向け | 洗練された文章。絵文字・ラフな表現はNG |
| タップル | 若年層向け | カジュアルでOK。趣味共通性を前面に |

## 添削フィードバックテンプレート

```
【自己紹介文 添削フィードバック（下書き）】

■ 全体の印象
{全体的な評価を1〜2文で}

■ 修正が必要な箇所
{該当箇所の引用 → 修正案の提示}

■ 良かった点
{そのまま使えるポイント}

■ 添削後の全文
---
{添削後の自己紹介文全文}
---

■ 総合判定
{OK / 一部修正 / 大幅修正推奨}
```

## りょうコーチの添削実例ポイント（要約）

1. **冒頭を「アプリを始めた理由」に変える**（岡さん事例）
   - Before：「気になった方に足跡付けてます🐾」
   - After：「友人がこのアプリで素敵な彼女を見つけたので、羨ましいなと思い始めました。」

2. **セクション順を変える**（岡さん事例）
   - Before：性格→仕事→趣味  
   - After：仕事→趣味→性格（仕事を最初に置くと信頼感が高まる）

3. **受け身な末文を削除する**（岡さん事例）
   - 「足跡は追いきれてません、気軽にいいねしてください」→ 削除

4. **OKと判断したらそのまま通す**（ひろさん事例）
   - 添削不要なら「OK、このまま行きましょう！」と明示

## エスカレーション基準

- テキストを送ってもらえれば自動で添削下書きを生成可能
- アプリ名が不明な場合は確認する
- 大幅修正が必要な場合はりょうに確認を回す
