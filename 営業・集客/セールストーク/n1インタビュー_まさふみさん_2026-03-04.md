## セールスへの応用

### まさふみさん型（中級者・経験者）へのアプローチ

```

「コンサルを受けてもなぜか変わらない。

「急ぎで相談したい時に相談できない。

```

### 他サービス経験者へのクロージングフレーズ
```
「前のコンサルで変わらなかったのは、あなたのせいじゃない。

電話音声・LINE実例など『正解』が見える点と、
タイムリーに相談できる体制があること。
やることは一緒でも、見える景色が全然違います。」
```

### 「最後のチャンス感」を引き出す問いかけ
```
「今何歳ですか？20代の残り時間、大切に使いたくないですか？
今の状態で30代を迎えたら、どうなってると思いますか？」
```

### 正解例・タイムリーフィードバックの強調（差別化ポイント）

セールスで必ず触れる：
- 電話音声・LINE実例などの「正解例」が見れること
