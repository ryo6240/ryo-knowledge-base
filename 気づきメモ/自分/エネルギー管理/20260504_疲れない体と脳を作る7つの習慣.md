体力は才能ではなく設計するもの。疲れない人は「仕組み」を持っているだけ。慢性疲労の89%は医学的に健康であり、体の問題ではなく「使い方」の問題。

## 7つの習慣（要点）

### 1. 呼吸（IAP呼吸法 ＋ 4:8の法則）
- 1日2万回の呼吸が浅いと酸素不足・集中力低下・疲労蓄積
- **IAP呼吸**：吸う時も吐く時もお腹を膨らませたまま → 体幹が安定し脳のエネルギー消費を減らす
- **4:8の法則**：鼻から4秒吸い、口から8秒吐く → 副交感神経優位・疲労物質の代謝が約1.7倍に
- 実践タイミング：① 朝起きた直後5回 ② 会議・重要作業の直前 ③ 集中力が切れた瞬間（コーヒーより先に10回）

### 2. 休息（90分サイクル ＋ パワーナップ）
- **ウルトラディアンリズム**：脳は90分周期で集中→急速を繰り返す
- 疲れる前に**90分ごとに5〜10分のマイクロレスト**を挟む
- **パワーナップ26分**：NASAの研究で注意力34%・反応速度54%向上。26分超は睡眠慣性が発生するので注意
- **コーヒーナップ**：仮眠直前にコーヒー → 目覚め頃にカフェインが効いて相乗効果
- 「疲れたら横になる」は最悪。脳の老廃物は血流増加で洗い流される → **アクティブレスト**（20分散歩・軽いストレッチ）が正解

### 3. 水分（定期的な少量補給）
- 体重の2%の水分が失われるだけで認知機能が約20%低下
- 喉が渇いた時点で1〜2%の脱水がすでに始まっている（警告サイン）
- **30〜60分ごとに150〜200mlをこまめに**補給。一度に大量摂取は非効率
- 午後の疲労感が最大47%軽減されるデータあり
- デスクに常温水ボトルを置く・30分ごとのリマインダー設定が有効

### 4. 食事（血糖値スパイク対策）
- 白い炭水化物（ラーメン・白米・パスタ）→ 血糖値スパイク → インスリン過剰 → 血糖値急落 → 強烈な午後の眠気・集中力低下
- 対策①：**ベジタブルファースト**（食事の最初に野菜・キノコ・食物繊維を食べる）
- 対策②：**GI値の低い食品に置き換え**（白米→玄米・食パン→全粒粉・うどん→そば）
- 発想の転換：「お腹が空いたから食べる」→「細胞に安定した燃料を届ける行為」

### 5. 睡眠（最初の90分が全てを決める）
- 睡眠の質を決めるのは量より**最初の90分の深いノンレム睡眠**
- この黄金の90分で：成長ホルモン分泌・細胞修復・免疫強化・脳内老廃物の排出
- アルコールは入眠を早める一方で最初の90分の質を著しく低下させる
- **起床時間を365日固定**（就寝時間は多少ずれてOK）。週末に2時間以上遅く起きると「社会的時差ボケ」が発生
- 就寝前の3習慣：① 寝る1〜2時間前からブルーライトを避ける ② 室温18〜20℃に設定 ③ 毎晩同じルーティン（ストレッチ・読書・瞑想・日記）

### 6. 脳のケア（情報断食 ＋ 耳マッサージ ＋ 完全オフ日）
- 現代人の疲労の8割は**脳のオーバーヒート**（情報処理過多）
- **情報断食**：朝起きて30分スマホを開かない / 食事中スマホ裏返し / 就寝前1時間SNS禁止 / 午前中1〜2時間の通知全オフ
- **耳マッサージ**：両手で耳をつまみ上・下・横・斜め方向に各10秒引っ張る。自律神経に作用し交感神経が緩む。どこでも30秒でできる
- **週1日の完全オフ**：タスク・メール・SNS一切なしの日を意図的に作る → 翌日の集中力・判断力・創造性が著しく向上

**【なぜそうなるか（構造）】**
7つの習慣は独立したテクニックではなく相互作用するシステム。呼吸が整うと睡眠の質が上がり、睡眠が上がると脳の処理能力が回復し食欲コントロールも楽になる。水分が十分だと集中力が上がり食事の選択も変わる。1つ始めると他が動き出す。

**【どう使うか（応用）】**
- サロン生の「やる気が出ない・疲れやすい」悩みへの実践的アドバイスとして使える
- 「体力は才能ではなく設計」→「モテも同じ構造」という文脈でコーチング導入に活用
- 特に**ベジタブルファースト・水分補給・パワーナップ26分**は即日実践できる提案として使いやすい
- りょうさん自身のパフォーマンス管理：朝の4:8呼吸・デスクの水ボトル設置
