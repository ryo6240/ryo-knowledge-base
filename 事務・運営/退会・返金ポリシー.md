# 退会・返金ポリシー

> 出典：MRKサロン コンサルティング契約書（スタンダード・ライト）

---

## クーリングオフ（第5条）

| 項目 | 内容 |
|------|------|
| 期間 | 契約締結日から **8日以内** |
| 方法 | 書面による申請 |
| 効力発生 | 解除する旨の書面を発信したときから |
| 返金 | 支払い済み代金を **速やかに全額返還** |
| サービス費用 | 既に提供したサービスの対価は請求しない |
| 違約金 | 請求しない |
| 損害賠償 | 請求しない |

## 途中解約（第6条）

**原則として、途中解約はできません。**

> 「正当な事由なく甲の事情において本契約の解除をすることはできない。」

- 契約期間：**初回写真撮影日から6ヶ月間**
- 正当な事由がある場合は、甲乙双方で誠意をもって協議（第7条）

## CSエージェント向けメモ

| シーン | 対応方針 |
|--------|---------|
| 「返金してほしい」 | 契約から8日以内かどうか確認。8日以内ならクーリングオフ案内。8日超の場合は**即りょうさんへエスカレーション** |
| 「退会したい」 | 原則不可である旨を伝え、**即りょうさんへエスカレーション** |
| 「追加料金が発生するか不安」 | 第3条より追加請求はない旨を案内してOK |
| 「契約期間はいつまでか」 | 初回写真撮影日から6ヶ月間と案内してOK |

> ⚠️ 返金・退会に関する話は必ずりょうさんへエスカレーション。CSが単独で判断・回答しない。
