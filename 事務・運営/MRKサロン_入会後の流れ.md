# MRKサロン 入会後の流れ

> 出典：⭐️MRKサロン入会後の流れ（Google Docs・案内資料）
> 事務タスクフロー④「Discordへ招待」時にサロン生へ送付するドキュメント

MRKサロンにご入会いただきありがとうございます。
入会〜初回面談まで、以下の5つのステップを順番に進めてください。

---

## ① 「入会後アンケート」への回答

現在地を把握し、最適なサポートを行うための大切なアンケートです。「今の正直な気持ち」をありのままに回答してください。

- **回答期限**：入会24時間以内
- **回答フォーム**：https://docs.google.com/forms/d/e/1FAIpQLSeRxixt-dlRDAvWt0D5okeX8nZCZRr8a1YWgFuZr9XVgFYF7Q/viewform

---

## ② 公式Discordへの登録

MRKサロンの活動拠点（メインコミュニティ）はDiscordです。最新情報のキャッチアップや活動報告はすべてここで行います。

- **登録説明動画**：https://youtu.be/vjmV3Gc5lQE
- **招待リンク**：別途、個別メッセージにてお送りします

**【重要：設定ルール】**アカウント名は、他のメンバーが認識しやすいよう以下の形式で設定してください。

```
表示名：LINE名 +（年齢）
例：りょう (30)
```

---

## ③ Discordでの自己紹介

Discordへの登録が完了したら、まず「**自己紹介チャンネル**」へ投稿しましょう。仲間への挨拶とともに、自分の目標を言語化することでモチベーションを高めます。

### 自己紹介フォーマット（コピーして使ってください）

```
1. 年齢：
2. 職業：
3. 居住地：
4. 過去・現状の自己分析：▶（例：これまでマッチングアプリで苦戦してきた、自分に自信を持ちたい等）
5. サロン活動を通じてどうなりたいか？：▶（例：3ヶ月以内に彼女を作る、圧倒的に清潔感のある男になる等）
6. 意気込み・挨拶：▶
```

> ※ 自己紹介の投稿が確認できましたら、運営より「**個別チャットルーム**」へ招待し、初回面談の日程調整を行います。

---

## ④ 目標設定シートの記入

各サロン生ごとに専用の「**管理シート（スプレッドシート）**」を作成します。このシートは、自身の成長を記録するロードマップになります。

まずはシート内の「初回面談」に関連する項目（**黄色セル**）の記入をお願いします。

### 目標設定シートの記入項目

| 項目 | 内容例 |
|------|--------|
| 名前 | けいと |
| 年齢 | 27歳 |
| 誕生日 | 1997/02/25 |
| 活動エリア | 東京（新宿・新大久保・渋谷・中目黒） |
| コース | スタンダード |
| サポート期間 | 2025/02/18〜2025/08/17 |
| 撮影日 | 2025/02/18 @ 虎ノ門 |
| アプリ開始日 | 2025/02/22 with スタート |
| **1年後の目標** | 理想の彼女像・その横にいる男性像を具体的に記入 |
| **現状は？** | 課題・悩み・今できていないことを正直に記入 |
| **目標達成に必要なこと** | 具体的な行動・習慣・取り組みを記入 |
| **数値目標** | 電話：12回/月、デート：8回/月 など |

> ※ 初回面談までの期間、サロン内のコンテンツを先行してインプットしていただいて構いません。優先的に取り組んでほしい内容がある場合は、個別にご連絡します。

---

## ⑤ 初回面談（キックオフ）

いよいよ本格的な活動のスタートです！現状の分析を行い、理想の未来へ向けた「専用のアクションプラン」を策定します。

- **内容**：目標設定のセットアップ、サロンの活用ルール説明、今後の進め方相談
- **形式**：オンライン個別面談

---

以上のステップを完了させることで、迷いなく活動をスタートできます。わからないことがあれば、いつでも運営までお気軽にご連絡ください。

**最高の結果を出すために、これから一緒に頑張っていきましょう！**

## CSエージェント向けメモ

| シーン | 対応方針 |
|--------|---------|
| 入会後24時間経過、アンケート未回答 | アンケート回答を促すメッセージを送る（りょうさん確認のうえ） |
| 自己紹介が投稿された | 個別チャットルームへ招待し、初回面談の日程調整を行う |
| 目標シートの書き方がわからない | 黄色セルのみ記入すればOKと案内。例を参考に促す |
| 「何から始めればいいですか？」 | このドキュメントのリンクを送付し、①から順に進めるよう案内する |
