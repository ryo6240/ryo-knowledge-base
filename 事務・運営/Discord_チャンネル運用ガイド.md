## 個別チャンネル

### 概要
- **作成タイミング**：自己紹介チャンネルへの投稿確認後、運営から招待して作成
- **用途**：日常の質問・相談、個別フィードバック、週報投稿、プチ報告
- **表示名ルール**：`LINE名 +（年齢）`　例：`りょう (30)`
- **退会・サポート終了後**：チャンネルはそのまま残す（削除しない）
- **閲覧権限**：サロン生は自分の個別チャンネルのみ閲覧可。他のサロン生のチャンネルは見られない

### 週報の流れ
1. 毎週日曜日にりょうさんが「全体共有」チャンネルで提出アナウンスを投稿
2. 各サロン生が個別チャンネルに週報を投稿（タイミングは人によりバラバラ）
3. りょうさんがコメントを返す

> 週報フォーマットは「週報フォーマット」チャンネル（ID: `1463556600550654205`）を参照

## 禁止事項

サロン内の投稿・やり取りに関して、以下は禁止です。

- **スクショの外部共有**：チャンネル内のやり取りをSNSや外部に持ち出すこと
- **個人情報の共有**：他のサロン生や女性の個人情報をチャンネルに投稿すること

## CSエージェント向けメモ

| シーン | 対応方針 |
|--------|---------|
| 自己紹介が投稿された | 個別チャンネルへ招待・初回面談の日程調整（りょうさん確認のうえ） |
| 週報が投稿された | **りょうさん確認必須**。CS単独でコメントしない |
| デート報告に返信がない | 放置時間を確認し、りょうさんへエスカレーション |
| 成果報告が投稿された | 対談動画撮影の提案タイミングをりょうさんへ共有 |
| 入会後24時間以上・自己紹介未投稿 | リマインドを検討（りょうさん確認） |
| スクショ・個人情報の外部共有を発見 | 即りょうさんへエスカレーション |
