# 日記処理プロンプト

音声データや口頭メモをこのプロンプトと一緒にClaudeに渡すと、日記ファイルを自動生成・整理します。

---

## 使い方

1. 音声データをテキスト化したもの（文字起こし）を用意する
2. Claudeに以下のように依頼する：

```
下記の音声文字起こしを日記として整理してください。

【文字起こし】
（ここに貼り付け）
```

---

## Claudeへの処理指示（内部用）

以下のルールで処理してください：

### 保存先
- 日記本体: `/Users/kawairyouhei/ナレッジ/事務・運営/日記/YYYY/MM/YYYY-MM-DD.md`
- テンプレート: `/Users/kawairyouhei/ナレッジ/事務・運営/日記/_テンプレート.md`

### 処理ルール

1. **テンプレートに沿って構造化**する
   - スケジュール・数字・エネルギーログ・学び・アイデア・翌日タスクの8項目に分類
   - 話し言葉はそのまま活かし、意味が変わらない程度に整える

2. **コンテンツアイデアが含まれる場合**
   - 日記の「6. コンテンツアイデア」に記載する
   - 内容が具体的なら `/Users/kawairyouhei/ナレッジ/コンテンツ/` への転記を提案する

3. **サロン生の固有名詞・個人情報が含まれる場合**
   - 日記には記載してよい
   - 顧客管理ファイルへの転記が必要な場合は確認を取ってから行う

4. **外部記事・URLへの言及が含まれる場合**
   - ジャンルを判断して `/Users/kawairyouhei/ナレッジ/コンテンツ/外部参考資料/` への追加を提案する

5. **ビジネス数字が含まれる場合**
   - テーブルに整理して記録する
   - 月末には `/Users/kawairyouhei/ナレッジ/経営ダッシュボード/` への集計を提案する
