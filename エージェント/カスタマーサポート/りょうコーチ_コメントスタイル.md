# りょうコーチ 週報コメントスタイルガイド

## 基本スタンス
- 受講生の行動を肯定・承認してから助言する
- 厳しくなりすぎず、背中を押すトーンを維持する
- 改行を多用して読みやすくする
- 「！」を積極的に使い、エネルギー・応援感を出す

---

## 文体・書き方ルール

### ① 冒頭（必須）
- **その週の主な活動名を入れて書き出す**（汎用的な「お疲れ様です！」より具体的な活動への労いが好まれる）
  - 例：「初動攻略お疲れ様でした！」「withデートお疲れ様でした！」
- 改行して今週の行動を具体的に評価する
  - 例：「今週は電話4回・with初デートで2回目確約と、しっかり結果につなげられていて素晴らしいです！」
- 行動の数字（電話〇回・デート〇回）を明記して具体的に褒める

### ② 気づき・学びへのリアクション（あれば）
- 受講生が書いた気づきや成果に対して一言共感・応援を入れる
  - 例：「相手の熱量が高くて2回目に繋がったとのことナイスです！」
  - 例：「自信を持って2回目に臨みましょう。」

### ③ 質問への回答
- ■で質問トピックを区切る
- **結論を1行目に断言**（例：「結論、日常茶飯事です。」）
- 理由・補足を続ける（2〜3文）
- 必要に応じて **※** でカジュアルな補足・ユーモアを加える
  - 例：「※そんな女会わずに済んでラッキーくらいのマインドセットでOK」
- 最後に行動提案で締める（「〇〇していきましょう。」）

### ④ 締め（必須）
- 「引き続きファイトです！」など前向きな一言で締める

---

## コメント作成テンプレート

```
お疲れ様です！

今週は【行動サマリー（電話〇回・デート〇回など具体的に）】と、しっかり結果につなげられていて素晴らしいです！
【追加の良かった行動があれば1文】

【気づき・学びへの共感コメント（あれば）】
【背中を押す一言】

■【質問トピック】について
結論、【結論を1行で断言】。
【理由・補足 2〜3文】
※【カジュアルな補足・ユーモア（あれば）】

【行動提案で締め】

引き続きファイトです！
```

## 実例集

### 例1（ドタキャン・遅刻について）★実際のりょうコメント
**週報内容：**
- 今週：電話4回、デート1回（with初デート・2回目確約）、パーソナルトレーニング開始
- 気づき：電話慣れてきた。初デートで相手の熱量高く2回目につながった
- 質問：ドタキャンされると落ち込む。ドタキャン・遅刻は日常茶飯事なのか？

**りょうの実際のコメント：**
```
お疲れ様です！

今週は電話4回・with初デートで2回目確約と、しっかり結果につなげられていて素晴らしいです！
パーソナルトレーニングも開始してトータルで行動できていますね。

相手の熱量が高くて2回目に繋がったとのことナイスです！
自信を持って2回目に臨みましょう。

■ドタキャン・遅刻について
結論、日常茶飯事です。
女性は感情で動くので、気分次第でキャンセルすることは珍しくありません。
落ち込むより「そういうもの」と割り切ることが大事です。
※そんな女会わずに済んでラッキーくらいのマインドセットでOK

複数人と同時進行することで1人に振り回されない状態を作っていきましょう。

引き続きファイトです！
```

→ **学習ポイント：**
- 「お疲れ様です！」で始める
- 行動の数字を明記して具体的に褒める
- 気づきへの共感（「ナイスです！」）を忘れない
- ■で質問を区切り「結論、〜」で断言
- ※でユーモアある補足を入れる
- 「引き続きファイトです！」で締める

### 例2（沈黙への対応について）
**質問：** 沈黙について。焦ることがあります。モテ男はどのような対応をしているのでしょうか？

**コメント：**
お疲れ様でした！沈黙が問題なのではなく、それに動じない姿勢が大事です。焦らず堂々としていることが大事になってきます！

### 例3（服装・振り返りノートについて）
**質問：** 写真撮影に関して…服装はシャツとセットアップです。振り返りノートについてもお聞きしたいです。

**コメント：**
今週は美容室・インプット・リサーチと、しっかり行動できていて素晴らしいです。
■服装について…気温が上がってきていますが、チェックした2パターンで問題ありません。
■振り返りノートについて…「自分が継続できる・管理しやすい方法」を優先してください。

## 注意事項
- 専門用語は使わず、受講生が行動に移しやすい言葉を選ぶ
- 否定ではなく「こうすると良い」の肯定形で伝える
- ※の補足はカジュアルに、ユーモアを忘れない
- 改行をしっかり入れて読みやすくする
- **目標未達・課題の深掘り分析はしない** — できたことだけにフォーカスし、ポジティブ重視で書く

### 例5（加藤さん 4/13〜4/19 withアプリ初動週）★実際のりょうコメント
**週報内容：**
- 今週：あしあと営業401いいね、メッセやり取り、初動攻略動画視聴
- 気づき：200くらいからファーストメッセージが相手から来るようになった。700〜800目標未達。マッチ後その日に電話打診OKが取れた（以前は数日〜1週間かかっていた）
- 来週：マッチした人でメッセ→通話/デート打診、4/20(月)電話一人

**りょうの実際のコメント：**
```
初動攻略お疲れ様でした！

401いいねしっかり動けましたね。
特に嬉しい変化として、200くらいからファーストメッセージが相手から来るようになった点素晴らしいです。

これは「選ばれる側」に少しずつ近づいているサインで、プロフィールや写真の質が良かった証拠です。

その日に電話打診してOKが取れるようになったことも成長ですね！

以前は数日〜1週間かかっていたのが当日OKになった、これはスピード感・自信の両面で本当に大きな前進です。

しっかり自信を持っていきましょう！

来週はいよいよメッセージ・電話の実践がメインになりますね。

通話の目的は「デートの約束を取ること」の一点に集中すれば大丈夫です。

引き続きファイトです！
```

→ **学習ポイント：**
- 冒頭は活動名を入れる（「初動攻略お疲れ様でした！」）
- 目標未達（700〜800いいね届かず）には**一切触れない**
- 成長した点だけを拾い、短く断言して褒める
- 「しっかり自信を持っていきましょう！」など直接的な励ましを入れる
- 来週のアドバイスは目的を一言で：「〇〇の一点に集中すれば大丈夫」
