# 自己紹介文 添削基準

> 用途：カスタマーサポートエージェントが自己紹介文の添削下書きを生成するための基準  
> 出典：りょうコーチ添削実例（岡さん・けいさん・ひろさん）  
> 出典：りょうコーチ添削実例（岡さん・けいさん・ひろさん・竜司さん・タクマさん・フミアキさん・徳留さん・優希さん・しんいちさん・加藤さん）  
> 最終更新：2026年4月22日

---

## 添削の基本方針

1. **冒頭文でアプリを始めた理由を自然に書く**（定番：「友人がこのアプリで素敵な方を見つけていて、羨ましいなと思い始めました！」）
2. **受け身・消極的な表現を削除する**
3. **セクション順は「仕事→性格→趣味→恋愛観」が基本**
4. **「足跡追いきれてません」「まずはメッセージからよろしくお願いします」などの受け身表現はNG**
5. **全体的にシンプル・自然体で書く**
6. **末文に電話誘導フレーズを追加する**（「メッセージだと分からないことも多いので、電話をしてからお会いしたいです。」）
7. **締め文は定型フレーズに統一する**（「少しでもいいなと思ったら / 気軽にいいねお願いします👍🏻」）

## セクション別チェック基準

### 冒頭文
| OK | NG |
|----|-----|
| 「友人がこのアプリで素敵な方を見つけていて、羨ましいなと思い始めました！」 | 「気になった方に足跡つけてます」 |
| 「堅苦しく婚活というより、まずはラフに話してみて〜」（個性的導入） | いきなり自己PRから始まる |
| 「素敵な出会いに巡り合えたらと思い始めました！」 | 「プロフィールを見ていただきありがとうございます！」のみ |
| 前向きな一言で締める | 「初めまして！〇〇といいます。」で始まる（名乗りから始めない） |
| プロフィール直後に個性的な一言（猫と暮らしている等） | 身長/MBTI/肩書き（「最後の恋人」等）を冒頭に書く |

### 仕事・職業
| OK | NG |
|----|-----|
| 職種＋やりがいを1〜2文 | 「都内で働いています」（曖昧すぎる） |
| 具体的・信頼感のある書き方 | 職場名を細かく書きすぎる |
| 「なぜその仕事を選んだか」のきっかけを1文 | 職種だけで終わる |
| スキルアップへの取り組みを書く（向上心をアピール） | 現状のみ記述で成長感がない |

### 趣味・好きなこと
| OK | NG |
|----|-----|
| 共通の話題になりやすいものを先頭に（カフェ→スポーツ→筋トレ等） | 趣味の羅列だけで感情がない（縦棒区切りで列挙） |
| 「なぜ好きか」を1文添える | マニアックすぎる趣味のみ |
| 各趣味に具体的なエピソードを1文（「鎌倉の豆柴カフェが最高でした」等） | 抽象的な説明のみで個性がない |
| 趣味ごとにセクション分け（・カフェ、・料理、・運動 等） | 趣味をひとまとめのブロックに詰め込む |

### 性格
| OK | NG |
|----|-----|
| 具体的なエピソードや特徴 | 「穏やかです」だけで終わる |
| 第三者から言われることを引用（「〜と言われます」） | 自己申告が強すぎる |
| 行動力を示す一言（「思い立ったらすぐ行動するタイプです」） | 性格の羅列のみで印象が弱い |

### 好きなタイプ・恋愛観
| OK | NG |
|----|-----|
| 相手が「自分かも」と思えるような描写 | 「笑顔が素敵な人」（抽象的すぎる） |
| お互いを尊重する表現 | 「理想が高すぎる」と思われる表現 |

### 末文・締め文
| OK | NG |
|----|-----|
| 電話誘導：「メッセージだと分からないことも多いので、電話をしてからお会いしたいです。」 | 「足跡は追いきれてません」 |
| リアル誘導：「気が合えばまずはカフェでお茶しましょう」 | 「まずはメッセージからよろしくお願いします！」（受け身） |
| 定型締め：「少しでもいいなと思ったら / 気軽にいいねお願いします👍🏻」 | 「いいねください」「マッチしましょう」（積極的すぎる） |
| 「ここまで読んでいただきありがとうございます！」は削除 | 長々とした感謝の言葉で締める |

## アプリ別 注意点

| アプリ | 特徴 | 注意点 |
|--------|------|--------|
| with | 比較的カジュアル | 絵文字少量はOK。明るい文体 |
| Pairs | 真剣度が高い | 誠実さ・落ち着いた文体。絵文字控えめ |
| 東カレデート | ハイスペック層向け | プロフィール文に絵文字NG。洗練された文章。メッセージでは相手に合わせる |
| タップル | 若年層向け | カジュアルでOK。趣味共通性を前面に |

{OK / 一部修正 / 大幅修正推奨}

## 削除すべき表現（NG表現一覧）

| NG表現 | 理由 |
|--------|------|
| 「足跡は追いきれてません、気軽にいいねしてください」 | 受け身・消極的 |
| 「まずはメッセージからよろしくお願いします！」 | 受け身・自らアクションしない印象 |
| 「ここまで読んでいただきありがとうございます！」 | 不要な丁寧語。末文が間延びする |
| 「プロフィールを見ていただきありがとうございます！」 | 冒頭に不要。アプリを始めた理由に差し替える |
| 「初めまして！〇〇といいます。」 | 冒頭に不要。名乗りから始めない |
| 身長/MBTI/肩書き（「最後の恋人」等）を冒頭に書く | 自己PR感が強く、冒頭に置く必要がない |
| 愚痴・ネガティブ表現（「毎日8時間も座りっぱなし…」） | 印象が暗くなる |
| 「不当な人間関係を経験しましたが〜」等の過去の苦労話 | ネガティブな印象を与える |
| 趣味を縦棒区切りで羅列（「カフェ｜ボクシング｜ねこ｜…」） | 個性が伝わらない。セクション分けして深掘りする |

## 特殊事情の記載ルール

| 事情 | 対応方法 |
|------|----------|
| 吃音症・話し方の特徴 | 削除せず【最後に】セクションの末尾に自然な表現で記載（「ゆっくり話せばしっかりお話できます」等） |
| 過去の職歴（転職・経営経験等） | 仕事セクションに「〇年続けた後〜」と具体年数を添えて記載 |
| 居住エリア（高級感のある場合） | 日常セクションに自然に追加（「銀座・有楽町エリアに行くことが多いです」等） |

## りょうコーチの添削実例ポイント（要約）

1. **冒頭を「アプリを始めた理由」に変える**（岡さん事例）
   - Before：「気になった方に足跡付けてます🐾」
   - After：「友人がこのアプリで素敵な彼女を見つけたので、羨ましいなと思い始めました。」

2. **セクション順を変える**（岡さん事例）
   - Before：性格→仕事→趣味  
   - After：仕事→趣味→性格（仕事を最初に置くと信頼感が高まる）

3. **受け身な末文を削除する**（岡さん事例）
   - 「足跡は追いきれてません、気軽にいいねしてください」→ 削除

4. **OKと判断したらそのまま通す**（ひろさん事例）
   - 添削不要なら「OK、このまま行きましょう！」と明示

5. **仕事のきっかけ・向上心を書く**（けいさん事例）
   - 職種だけでなく「なぜその仕事を選んだか」「スキルアップへの取り組み」まで書くと印象が深まる
   - After例：「始めたきっかけは〜。その為に今は〜の勉強をし、スキルアップを目指してます！」

6. **趣味に具体的エピソードを添える**（けいさん事例）
   - 趣味の羅列でなく、各趣味に1文のエピソードを入れると個性が出る
   - After例：「鎌倉の豆柴カフェが最高でした。」「先日は福岡へ弾丸旅行に行ってきました！」

7. **末文に電話誘導を追加する**（竜司さん・タクマさん・徳留さん・優希さん・しんいちさん事例）
   - 「メッセージだと分からないことも多いので、電話をしてからお会いしたいです。」を追加することでリアルへの動線を作る

8. **冒頭のNG情報を削除する**（優希さん・タクマさん事例）
   - 冒頭の「177cm/ENFJ/最後の恋人」「東京の府中市に住んでる」等は削除。アプリを始めた理由に差し替える

9. **趣味の順番を変える**（しんいちさん事例）
   - Before：筋トレ→スポーツ観戦→カフェ巡り
   - After：カフェ巡り→スポーツ観戦→筋トレ（共通の話題になりやすいものを先頭に）

10. **ネガティブ・愚痴表現を削除する**（優希さん・フミアキさん事例）
    - 「毎日8時間も座りっぱなし…目がしょぼしょぼ」「不当な人間関係も多く経験しました」→ 削除

11. **性格の「面白い」は自己申告しない**（徳留さん事例）
    - 「優しい」「面白い」→「優しい」のみに絞る（面白さは行動・エピソードで見せるもの）

## エスカレーション基準

- テキストを送ってもらえれば自動で添削下書きを生成可能
- アプリ名が不明な場合は確認する
- 大幅修正が必要な場合はりょうに確認を回す
