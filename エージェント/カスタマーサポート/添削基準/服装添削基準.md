# 服装 添削基準

> 用途：カスタマーサポートエージェントが服装の添削下書きを生成するための基準  
> 出典：FAQ実例・週報コメント  
> 最終更新：2026年5月1日

---

## 服装添削の基本方針

- **清潔感・サイズ感・色合い**の3点が基本
- マッチングアプリ写真用 or デート当日用かで視点が変わる
- 「モテる服装」より「減点されない服装」を最初に目指す

---

## りょうコーチのファッション哲学：体型補正が最重要

> ファッションで一番大切なのは「体型を補正できるか」。見た目の印象は体型補正で大きく変わる。

### 体型補正の4つのポイント

| ポイント | 目的 | 具体的な方法 |
|---------|------|------------|
| **①肩幅を大きく見せる** | 小顔効果・全体のバランス向上 | ドロップショルダー / 肩パッド入り / 横に広がるシルエットのアウター |
| **②腰の位置を高くする** | 足を長く見せる | ハイウエストパンツ / タックイン（シャツの前だけイン） |
| **③パンツと靴の色を合わせる** | 脚のラインを伸ばして足長効果 | 黒パンツ×黒靴、ネイビーパンツ×ネイビー系スニーカー など |
| **④アウターは丈を短めに** | 脚長効果・スタイルアップ | 腰丈〜ヒップ上のジャケット・ブルゾンが目安 |

これら4点を意識するだけで見た目の印象が大きく変わる。

## 服装チェック基準

| 項目 | OK | NG |
|------|----|----|
| サイズ感 | 体型に合ったサイズ（大きすぎず小さすぎず） | ダボダボ、きつすぎ |
| 清潔感 | シワなし、汚れなし、ピシッとしている | よれた服、毛玉、汚れ |
| 色 | ベーシックカラー（白・黒・紺・グレー・ベージュ） | 奇抜なカラー、主張の強い柄 |
| 組み合わせ | シンプルな配色（2〜3色以内） | 色が多すぎてうるさい |
| 季節感 | 季節に合った素材・重ね着 | 季節外れの服装 |

## 服装添削フロー（撮影前の実プロセス）

> 出典：けんごさん・古屋さん個別チャンネルの実際のやり取りより

1. **コーデ候補の写真提出**：サロン生が全身着用写真を複数枚送付（着用写真が推奨・判断しやすい）
2. **2パターン選定**：りょうコーチが候補の中から撮影用に2パターンを選ぶ
3. **シワの最終チェック**：当日に向けて「シワなどの最終チェックをお願いします」と指示
4. **アクセサリー確認**：服装選定後に「腕時計・ブレスレット・ネックレスはお持ちですか？」と別途確認
5. **当日持参**：選定した2パターン＋アクセサリーを撮影当日に持参

## りょうコーチの判断基準（Discord実発言より言語化）

> 出典：個別チャンネル（幹人さん・タクマさん・山田さん・古屋さん・ひろさん・shoheiさんほか）

---

### ① コーデ方向性の設定（個別最適化）

全員きれいめにするのではなく、**その人の雰囲気に合った方向性を最初に設定する**。

| 方向性 | 特徴 | 向いている人 |
|-------|------|------------|
| カジュアル系 | 素材感・遊びのある服 | 顔立ちが柔らかい・若く見える |
| キレイめ系 | テーラード・スラックス中心 | 大人っぽい・落ち着いた雰囲気 |
| 韓国系 | トレンド感・オーバーシルエット | スタイリッシュ・細身 |

**方針決定後はデータドリブンで修正する**：
> 「まずカジュアルで運用開始 → 反応を見て韓国系・キレイめに切り替える」（ひろさんch）

---

### ② 選定プロセス（断言形式）

- 候補を複数枚（全身着用写真）送ってもらい、**その中から2パターンを即決で選ぶ**
- 理由は長々述べず「**こちらでいきましょう！**」と画像添付＋断言形式で伝える
- 迷いの余地を与えない（受講生の判断負荷を下げるため）
- 問題なければ「大丈夫です！自信を持ってファイトです！」とメンタルサポートも添える

---

### ③ 色・インナーの基本ルール

- **初対面・撮影のインナーは白が基本**
  > 「初対面は明るい印象を与えたいので、インナーは白にしましょう！」（幹人さんch）
- 撮影時は**白Tと黒Tの両方**を持参させる（色違いで2パターン撮り分けるため）
- ベーシックカラー（白・黒・紺・グレー・ベージュ）を軸にする

---

### ④ アイテム別ルール（実例から）

**撮影用の服装指定例（タクマさんch 実際の指示文）：**
```
・アウター：黒シャツ、茶ジャケット（2パターン）
・インナー：白Tシャツ
・ボトムス：黒テーパードデニム（間に合わなければインディゴでもOK）
※寒ければ防寒用のアウター
```

### ⑤ アクセサリーの扱い

- **服装選定が終わった後に必ず確認する**
  > 「腕時計やブレスレット、ネックレスなどのアクセサリーはお持ちでしょうか？お持ちでしたらお送りください。」
- **あれば活用する。なければ無理に買わせない**（「あれば」「お持ちでしたら」という表現に注意）
- 選定後に「当日はこちらもお持ちください」と確定させる

## NGブランド・素材の傾向

### NGカテゴリ（系統・素材で判断）

| カテゴリ | 理由 |
|---------|------|
| ビッグロゴ・派手グラフィック系（ストリート過多） | 個性が強すぎて女性受けしにくい |
| テカテカしたポリエステル100%素材 | 安っぽく見える・清潔感が下がる。不自然な光沢が出る |
| Pコート | 現在は流行遅れ。着膨れしてスタイルが悪く見えやすい |
| ダッフルコート | 子供っぽい印象になりやすい。スマートさが損なわれる |
| 杢調（もくちょう）グレーニット | 着ている人が多い割に垢抜けない・田舎っぽい印象になりやすい |
| ダボダボのバルーン型パンツ・パジャマ系スウェット | だらしない・清潔感ゼロの印象 |
| ダメージ加工の強すぎるデニム | 清潔感を下げる |
| ヨレ・毛玉が目立つニット | 古さ・だらしなさに直結 |

### 素材のOK/NG目安

| 素材 | 判定 | 理由 |
|------|------|------|
| コットン（天然素材） | ○ | 清潔感・自然な風合いが出る |
| リネン（麻）薄手 | ○ | 夏の清潔感・季節感に◎。においも目立ちにくい |
| ウール（秋冬） | ○ | 上品・きれいめに見える |
| デニム（ダメージ少なめ） | ○ | 定番・合わせやすい |
| ポリエステル100%（テカあり） | × | 安見え・不自然な光沢が出る |
| ポリエステル混（テカなし） | △ | 素材感で判断。安っぽく見えなければOK |
| レザー（本革・フェイク問わず） | △ | 使いすぎると重くなる。1点使いまでが目安 |

## 体型別 おすすめシルエット

### 標準体型
- **方針**：サイズが合っていれば何でも似合う。フィット感のみ意識。
- **おすすめ**：ストレートパンツ / ワイドパンツ / フレアパンツ＋シンプルトップス
- **ポイント**：清潔感とサイズ感さえ押さえればOK

### 細身・華奢体型
- **方針**：肩幅を大きく見せることを最優先に意識する。レイヤードやゆったりめトップスで厚みを出す。
- **おすすめ**：ドロップショルダーのトップス / シャツ羽織り / タートルニット＋スリムパンツ
- **避けるべき**：全身ピタピタ（細さが強調されすぎる）/ 全身ダボダボ（体がなくなって見える）

### がっちり・筋肉質体型
- **方針**：ストレッチ素材でフィット感を活かす。シンプルな服で品よく見せる。
- **おすすめ**：ストレッチ素材のシンプルTシャツ＋スリムデニム / きれいめブルゾン
- **避けるべき**：パツパツのきつすぎるサイズ（服が主張する）/ ゆるすぎて体型が埋もれる

### ぽっちゃり体型
- **方針**：「Yラインシルエット」が基本。上にボリューム、下は細いパンツで縦長ラインを作る。ダークカラーで引き締める。
- **おすすめ**：Vネック＋テーパードパンツ / 着丈長めのシャツ羽織り＋黒スリムパンツ
- **テーパードパンツ**：腰回りにゆとりがあり裾に向かって細くなるため、体型カバーしながらスタイルよく見える万能アイテム
- **避けるべき**：横縞ボーダー / 白・黄などの膨張色の全身コーデ / オーバーサイズ全身（余計に大きく見える）

## 小物使いの詳細ルール

### アクセサリーの大前則

**まず1種類から始める。** ネックレス・ブレスレット・時計のどれか1つをコーディネートに取り入れるのが初心者の基本。慣れてきたらトータル2〜3点まで。ジャラジャラ重ね付けは女性受けが悪い。

### 時計

| 項目 | 推奨 | NG |
|------|------|-----|
| 文字盤 | シンプル・読みやすい | キャラクター・派手すぎるデザイン |
| バンド | レザー or メタルブレス | ラバー（スポーツ感が強い場合） |
| 大きさ | 38〜42mm程度 | 極端に大きい・小さい |
| ブランド感 | なくてOK | 偽ブランド品は絶対NG |

時計とブレスレットを両方つける場合、どちらもゴツいデザインだとやりすぎ感が出る。どちらかをシンプルにする。

### アクセサリー（ブレスレット・ネックレス）

- **シルバーブレスレット**：1〜2本まで。シンプルなチェーンorバングル推奨。女性ウケが最も安定している。
- **ネックレス**：細めのシルバーチェーンが無難。ペンダントは小さめにする。ゴツい鎖・太い金のネックレスはNG（安っぽい・ホスト感が出る）。
- **ゴールド系**：慣れないうちは避ける（重厚感・ホスト感が出やすい）。シルバーの方が誠実な印象になりやすい。
- **ピアス**：女性の評価が分かれるため、基本的には推奨しない。
- **つけすぎ禁止**：アクセサリーはトータル2〜3点まで。多すぎるとうるさい印象になる。

### バッグ

| タイプ | 判定 | 備考 |
|--------|------|------|
| キャンバストート（無地・シンプル） | ○ | 清潔感・デイリーに合う |
| レザートート・ショルダー | ○ | きれいめ・大人感 |
| バックパック（シンプル） | ○ | カジュアルシーンで◎ |
| キャラクター・ロゴ主張バッグ | × | 子供っぽく見える |
| ビニール袋・ボロボロのエコバッグ | × | 論外 |

### 靴

- **スニーカー**：白・黒・グレーなどシンプルカラー。汚れは絶対落とす。
- **レザーシューズ**：きれいめコーデに合わせて◎。
- **サンダル**：夏のカジュアルデートのみOK。足の清潔感も必須。
- **靴の状態**：かかとのすり減り・汚れはマイナス印象。定期的なメンテが必要。

### ベルト

- 靴の色と合わせるのが基本（黒靴→黒ベルト、茶靴→茶ベルト）。
- バックルはシンプルなものを選ぶ。
