# マッチングアプリ別戦略ナレッジ

サロン生が多く使うアプリはwithとPairs。それぞれ初動戦略が異なる。

---

## with 攻略法（2026年版）

### アプリの特徴
- マッチングアプリ中でアクティブユーザーが最多
- いいね数上限9,999＋で錯覚資産を積みやすい
- 登録後24時間以内の足あとに対して無料いいねが送れる
- 1日あたり16人無料いいね送信可能
- 心理テストで相性を事前確認できる
- フツメンでも成果が出やすい・真面目な層が多い

### 錯覚資産の仕組み
- 自分が確認できるプロフ：直近1ヶ月のいいね数
- 相手が確認できる画面：直近3ヶ月のいいね数＋マッチング数
- 「いいね！の多い順」検索で上位に表示されるため、いいね数が多いほど有利

### 最適な登録日時
**土曜日22時がベスト**（土日のログイン人数が最多。日曜登録はNG）

### 新規登録ブースト戦略（最重要）
登録後24時間以内に付いた足あとに対して無料いいねを送信し、いいね返しを獲得する。

**推奨ツール：ふむふむ君（月額1,188円、PC必須）**
- サロン生向けプロモコード：`JVVDHSHK`（1ヶ月無料）

**足跡営業の検索設定**
- 居住地：最寄りの主要都市（例：東京・神奈川・千葉・埼玉）
- 最終ログイン：オンライン（24時間以内）
- 並び替え：ログイン新しい順 / おすすめ順

**ふむふむ君設定**
- 足あと速度：0.5〜1.5秒
- 停止条件：1,000〜2,000人
- 定期リロード：500〜1,000人

### プロフィール設定

**基本情報**
| 項目 | 設定 |
|------|------|
| 身長 | 3〜5cm盛る（盛り靴の限度まで） |
| 体型 | 「普通」または「筋肉質」 |
| 居住地 | 地方の場合は最寄り都市を選択 |
| 年収 | 実額+1段階上 |
| タバコ | 「吸わない」 |
| 結婚意思 | 「良い人がいれば」 |
| 子ども希望 | 「はい」 |
| 家事育児 | 「二人で協力したい」 |
| デート費用 | 「男性が全て奢る」 |

**好みカード**：最低300枚、できれば1,000枚登録  
**心理テスト**：無料開催分は全て参加、初動で最低4個実施

### 有料会員
- 1ヶ月または3ヶ月プランを推奨（6・12ヶ月はNG）
- VIPプラン特典：既読確認・いいね多い順並び替え・フリーワード検索・毎月いいね20個追加
- **Web版から課金**（アプリより安い）

### 初日タイムスケジュール
1. 土曜22時：アプリ登録開始
2. 本人確認
3. プロフ設定（写真・自己紹介・詳細）
4. 好みカード・心理テスト登録
5. モテ度チェック100点化 → 運営へスクショ送信
6. Web版から課金
7. 足跡営業開始
8. 足あと返信者に無料いいね送信（1時間ごとが理想）
9. 日曜22時までに全員へのいいね完了

### 注意事項
- 24時間が勝負のためプロフ登録に時間をかけすぎない
- 初動中は相手からのいいね・マッチは一旦無視
- 退会から再登録まで1ヶ月必須

## Pairs 攻略法（2026年版）

### アプリの特徴
- 日本最大級（会員数2,000万人）
- 登録後3日間は「新着のお相手」として優遇表示
- 毎日無料いいね：アプリ8人＋Web10人＝計18人
- いいね数500+でカンスト → 錯覚資産が積みやすい
- 男性平均いいね数：25〜30

### 最適な登録日時
**金曜日19時がベスト**（3日間の優遇期間で金・土・日を網羅できる）

### ブースト戦略（最重要）
**効果：** 60分間優先表示 ＋ 24時間無料いいね権

**推奨スケジュール（合計10回）**
| 日時 | 回数 |
|------|------|
| 金曜21時〜24時 | 3回 |
| 土曜21時〜25時 | 4回 |
| 日曜20時〜23時 | 3回 |

**ブースト中の動き方：**「10分足跡営業 → 無料いいね送付」を1時間内で6回繰り返す

### プロフィール設定

**基本情報**（withとほぼ同じ原則）
| 項目 | 設定 |
|------|------|
| 身長 | 3〜5cm盛る |
| 体型 | 「普通」または「筋肉質」 |
| 居住地 | 地方なら最寄り都市（後で変更可） |
| 年収 | 実額+1段階上（上限：実額+200万円） |
| タバコ | 「吸わない」推奨 |
| 結婚意思 | 「良い人がいればしたい」 |
| デート費用 | 「男性が全て奢る」 |

**マイタグ：** 最低300枚、目標500枚登録（女性登録者10,000人以上のタグ優先）  
**ペアーズクエスチョン：** 10万人以上回答の質問を優先（検索露出UP）

### 有料会員
- **1ヶ月プランのみ推奨**（直近30日のいいね数表示のため長期は非効率）
- 料金：2,980円/月 ＋ ブースト用ポイント100pt（7,190円）
- **Web版から課金**（アプリより安い）

### 足跡営業
- ブースト前15〜30分から手動で開始
- フィルタ：最終ログイン「24時間以内」「ログイン順」
- **自動化ツール（ふむふむ君）はPairsでは規制が厳しいため手動推奨**

### 初日タイムスケジュール
1. 19時：アプリ登録
2. 本人確認
3. 写真・自己紹介・詳細プロフ設定
4. マイタグ・価値観・クエスチョン記入
5. Web版から課金（ブースト10回分）
6. 20時30分頃から足跡営業開始
7. 21時からブースト開始

### 注意事項
- 登録後30日経過まで再登録不可
- プライベートモードは不要
- ふむふむ君はwithには使えるがPairsでは非推奨

## タップル 攻略法（2026年版）

### アプリの特徴
- Pairsに次ぐ登録者数2位
- 20代前半ユーザーが多い・スワイプ型
- 写真と外見が良ければ攻略しやすい
- 気軽な出会いを求める女性が多い

### 攻略の最重要要素
**「写真 × アカウントスコア」**
- スコアは対女性評価（いいかも率）と対運営評価（アクティブ度・課金額）で構成

### 最適な登録日時
**金曜19〜22時**（月曜夜までの無料期間でスコアを上げてから課金）

### プロフィール設定
| 項目 | 設定 |
|------|------|
| 写真 | 全9枚埋める（顔2枚＋全身1枚＋ライフスタイル6枚） |
| 自己紹介文 | 最初の3行に強みを配置 |
| 身長 | 3〜5cm盛る |
| 年収 | 実額+1段階上 |
| 趣味タグ | 50個以上登録・タグベスト5項目全て埋める |

### 日々の運用
- 本日のピックアップを毎日30回使い切る
- スワイプは朝・昼・晩の3回以上に分けて実施
- チャンスタイムは必ず完了
- 課金後は受信メッセージに即対応

### 注意事項
- LINEなどSNSのIDを直接送信するのは禁止（通報リスク）
- 下ネタ・ヤリモク感のある表現もNG

## 東カレデート 攻略法（2026年版）

### アプリの特徴
- 入会審査あり（20代：年収500万以上 / 30代以上：750万以上）
- ハイスペック会員が多い
- いいねが無限に送れる
- いいね数が相手に可視化されない（デメリット）

### 入会審査
「会員審査」（24時間）→「運営審査」（1営業日）の2段階。正面顔がわかる写真と充実したプロフィールが重要。

### 最適な登録日時
**水曜19〜22時**（金〜日のゴールデンタイムに審査完了を狙う）

### スタート戦略（最重要）
**ふむふむ君（自動いいねツール）を活用**
- 新規登録から1週間は集中的にいいねを送信
- 月額1,188円、サロン生向けプロモコード：`JVVDHSHK`（1ヶ月無料）
- マッチ数はいいね送信数に比例

### プロフィール設定
| 項目 | 設定 |
|------|------|
| 写真 | 6枚（自分3枚＋ライフスタイル3枚） |
| 年収 | 実額+1段階上 |
| 身長 | 3〜5cm盛り可 |
| デート費用 | 「男性が全て支払う」 |
| 基本情報 | 全項目埋める |

### 検索条件
年齢（±10歳）と居住地のシンプル設定で十分。

### 有料プラン
- **1ヶ月（6,500円）のみ推奨**（複数月契約はいいね送信者が減少するため非推奨）
- プレミアムオプションは不要

### 運用サイクル
1日100人へいいね → 約10人マッチ → メッセージ・電話・デートへ進行。マッチが減れば再度いいね送信再開。
