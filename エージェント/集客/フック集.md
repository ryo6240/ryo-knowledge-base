# SNS投稿 ファーストフック集

> 目的：投稿の冒頭1〜2行で読者を止める。ターゲット（恋愛経験が少い20〜30代男性）の「これ俺のことだ」を引き出す。
> 出典：成果報告集・ターゲットペルソナ・りょうコーチフィードバック集より作成

---

## 共感型フック（「俺のことだ」と思わせる）

```
4年マッチングアプリをやり続けて、人生初彼女ができなかった男の話。
```

```
27年間、彼女ができなかった男が変わった理由。
```

```
15年半ぶりに彼女ができた35歳がやったこと。
```

```
デートは盛り上がった。でもその後LINEの返信が来なくなった。
なぜか分かりますか？
```

```
「いい人だね」と言われ続けて、付き合えたことは一度もなかった。
```

```
マッチングアプリ、頑張ってるのに全然マッチしない。
それ、やり方の問題じゃなくて「見せ方」の問題かもしれません。
```

```
社会人になって5年。気づいたら恋愛から完全に遠ざかっていた。
```

---

## 実績型フック（信頼・実証を示す）

```
サロン入会2ヶ月で人生初彼女。これが再現できた理由を話します。
```

```
累計140名・彼女GET率92%。この数字の裏側にある話。
```

```
マッチ数が0から月300件になった男がやったこと、1つだけ教えます。
```

```
「写真が9割」は本当だった。Before→Afterを見てください。
```

```
サロン生が1週間で人生初彼女をつくるまでの全記録。
```

```
電話2回・デート2回で付き合った男が意識したこと。
```

## 教育型フック（「知らなかった」を作る）

```
デート後に返信が来なくなる本当の理由、テクニックじゃないです。
```

```
マッチングアプリは「加点方式」じゃない。これを知らないと一生成果が出ない。
```

```
「行きたい場所に連れて行く」のが優しさだと思ってた人へ。
```

```
メッセージが正しくても彼女ができない理由。
```

```
自己流で頑張っている人が「再現性のない努力」に陥る構造。
```

```
写真・メッセージ・会話。どれか1つだけ磨いても彼女ができない理由。
```

## 恐怖→希望型フック（行動を促す）

```
30代になっても「そのうち彼女できる」と思っていた人のリアル。
```

```
このまま何も変えなかったら、5年後の自分はどうなっているか。
```

```
「まだ若いから大丈夫」が一番危ない理由。
```

```
今動いた人と、迷い続けた人では3ヶ月後に何が変わるか。
```

## 格言型フック（りょうコーチの言葉から）

```
「相手が会いたいと思っているなら、24時間以内に必ず返信は来る。
返信が来ないのはLINEが悪いからではなく、
相手の中に"もう一度会う理由"が生まれていないだけ。」
```

```
「デートはサービスではなく、関係性を作る"設計"。」
```

```
「優しさと主導権はまったくの別物。」
```

```
「恋愛は才能でも顔でもなく、正解を知っているかどうかの差。」
```

---

## 使い方のポイント

- フックは「1〜2行で完結」させる。続きを読みたくなる問いかけや余白を残す
- X（Twitter）では1投稿目の冒頭、Instagramではキャプションの1行目に使う
- 同じフックを連続で使わない（最低2週間は間隔をあける）
- 実績型は数字を必ず入れる（「多くの人が」より「140名が」）
