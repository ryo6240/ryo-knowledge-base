# SNS投稿テンプレート集

> 目的：集客エージェントが投稿を提案する際の型。フック集と組み合わせて使う。
> 出典：成果報告集・ターゲットペルソナ・サービス詳細・りょうコーチフィードバック集より作成

---

## テンプレート① 実績型（成功事例の紹介）

**目的**: 信頼構築・具体性で「自分にもできるかも」と思わせる

```
【フック】
〇〇年間で人生初彼女ができた男がやったこと。

【本文】
彼は入会当初、△△という状態でした。
サロンで変えたのはたった1点です。

▶ □□□□□□

たったこれだけで、2ヶ月で結果が出ました。
正解を知っているか、知らないかの差です。

【CTA】
プロフィールのリンクから無料相談を受け付けています。
```

**成功事例から使えるエピソード例**:
- 「27年間人生初彼女（サロン入会後すぐ）」
- 「4年アプリをやって変われなかった男が2ヶ月で突破」
- 「電話6回・デート2回で外資系コンサル彼女」
- 「サロン2ヶ月→入籍まで至ったケース」

## テンプレート② 教育型（知識・気づきを提供）

**目的**: フォロワー増・保存数アップ・専門家としての信頼確立

```
【フック】
〇〇だと思っている人へ。それ、逆効果です。

【本文（箇条書きまたはスレッド形式）】
多くの人がやってしまうNG行動：
・△△
・□□
・××

正しくはこうです：
・◎◎
・◎◎

理由は「相手の感情が□□だから」です。

【締め】
覚えておいてほしいのは、
恋愛はテクニックより先に△△が大事だということ。

【CTA】
保存しておくと後で見返せます。
```

**教育テーマ候補（フィードバック集・ペルソナから）**:
- デート後に返信が来なくなる本当の理由
- 行きたい場所に連れて行くのがNGな理由（主導権の話）
- LINEは「正しいか」より「楽しそうか」で判断される
- マッチングアプリの「足切り構造」を知らないと一生成果が出ない
- 写真・メッセージ・デートは全部揃えないと穴あきバケツになる
- 交際初期にすり合わせるべき4つのこと

## テンプレート③ 共感型（ペインを言語化する）

**目的**: 潜在層への認知・「俺のことだ」でフォロー・シェアされる

```
【フック】
〇〇という状況、心当たりありませんか？

【共感パート】
マッチングアプリを頑張っているのに成果が出ない。
デートに行けても2回目に繋がらない。
「いい人だね」と言われ続けて終わる。

これ、あなたのせいじゃないかもしれません。

【問題提起】
うまくいかない原因のほとんどは、
「才能」でも「顔」でもなく、△△を知らないことです。

【希望】
正解を知って、正しく動けば変わります。
実際にMRKサロンでは累計140名が結果を出しています。

【CTA】
気になる方はプロフリンクから。
```

**ペイン軸（ターゲットペルソナから）**:
- マッチしない / タイプとマッチしない
- メッセージが続かない
- デートに誘っても断られる・ドタキャンされる
- 2回目に繋がらない・いい人止まり
- 告白で保留→フェードアウト

## テンプレート④ Before/After型（変化の可視化）

**目的**: 写真投稿・ビジュアルコンテンツと組み合わせる。Instagram向き

```
【フック】
写真撮影の前と後。マッチ数が△△倍になりました。

【Before】
・プロフィール写真：スマホ自撮り
・マッチ数：月〇件
・デート数：月〇件

【After】
・プロフィール写真：MRKコーチ同行プロ撮影
・マッチ数：月△件（〇倍）
・デート数：月△件

【一言コメント】
「写真が9割」とよく言いますが、本当でした。
変わったのは写真だけ。中身はそのままです。

【CTA】
MRKサロンでは写真撮影2回がサポートに含まれています。
詳しくはプロフリンクから。
```

## テンプレート⑤ 格言・一言型（X / Twitter向き）

**目的**: 拡散・引用リツイート・りょうコーチのブランディング

```
「〇〇〇〇〇〇〇〇。」

— MRKサロン りょうコーチ
```

**使えるりょうコーチの言葉（フィードバック集から）**:
- 「相手が会いたいと思っているなら、24時間以内に必ず返信は来る。返信が来ないのはLINEが悪いからではなく、相手の中に"もう一度会う理由"が生まれていないだけ。」
- 「デートはサービスではなく、関係性を作る"設計"。」
- 「優しさと主導権はまったくの別物。」
- 「文章は短くてもいいが、気遣いは削るな。画面の向こうには人がいる。」

## テンプレート⑥ 季節・タイミング訴求型

**目的**: 特定の時期に「今動くべき」という緊急性を作る

```
【フック（時期に合わせて差し替え）】
クリスマスまであと〇日。

【ターゲットの心理に刺す】
今から動き始めた人と、「そのうちやろう」と思っている人では
〇週間後に大きな差が生まれます。

【実績で信頼】
MRKサロンでは、入会〜初彼女まで平均1〜2ヶ月。
早い人は1週間で結果が出ています。

【CTA】
まずは無料相談から。プロフリンクをご覧ください。
```

## 投稿制作フロー

1. **テーマ選定** → ペルソナのどのフェーズ・どのペインに刺さる投稿か決める
2. **フック選択** → [フック集](フック集.md)から合うものを選ぶ or 新たに作る
3. **テンプレート選択** → 上記①〜⑥から型を選ぶ
4. **素材投入** → 成果報告集・フィードバック集・サービス詳細から具体エピソードを入れる
5. **CTA確認** → 誘導先（LINE公式 or 無料相談）を必ず入れる
6. **投稿後記録** → 反応が良かったものは集客ナレッジのコンテンツパターン欄に記録する
